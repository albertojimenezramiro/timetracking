Linux

Descarga para Linux (.tar.gz)

Descomprime el fichero donde quieras y aparecerá un directorio TimeTracking con el programa. Tus datos de usuario se escribirán en el directorio .timetracking dentro de tu home.

Windows

Descarga para Windows (instalador .exe)

Instala el programa donde quieras y aparecerá una nueva carpeta TimeTracking con el programa. Tus datos de usuario se guardarán en una carpeta .timetracking dentro de tu carpeta de usuario de Windows (C:\Users\ en los Windows modernos o en C:\documents and settings\ en algunos más antiguos).
